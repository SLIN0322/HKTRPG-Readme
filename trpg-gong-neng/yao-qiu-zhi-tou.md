# 要求擲骰

![使用示範](<../.gitbook/assets/image (26).png>)

### 要求擲骰功能

方便GM提示玩家需要進行的擲骰，支援角色卡，所以可以調用.ch功能。

輸入指令後，會彈出按鈕，點擊後就會自動進行擲骰。

### 使用方法

`.re 要求擲骰`\
例如 .`re 1d100 哈哈, 1d3 SC成功, 1d10 SC失敗`

每個擲骰方法需要使用逗號(,)進行分割

{% hint style="info" %}
注意: 本功能支援Discord斜線(/)功能，只需要把(.)轉成(/)
{% endhint %}

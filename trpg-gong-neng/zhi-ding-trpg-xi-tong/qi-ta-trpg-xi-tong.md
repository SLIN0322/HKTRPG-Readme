# 其他TRPG系統



{% hint style="info" %}
由2022/05/11開始，過往使用BcDice的一些擲骰骰組取消，並以BcDice(.bc)功能取代。\
被取代如【朱の孤塔】【神我狩】【迷宮王國】【忍神】【DX2nd,3rd】 等等
{% endhint %}

* 以下指令詳情省略，輸入指令頭 + help 觀看詳情 eg: `.al help`
* 【魔女狩獵之夜】`.wn xDn+-y`
* 【命運Fate】 `.4df(m|-)(加值)`
* 【WOD黑暗世界】`.xWDy`
* 【貓貓鬼差】`.kc xDy z`

# Discord 修改舊訊息

{% hint style="info" %}
現在你可以修改由HKTRPG和webhook(角色扮演功能)所發出的訊息\
就像你自己只能修改自己的訊息一樣，此功能不能修改其他人或其他BOT的訊息。
{% endhint %}

`.edit`  Discord限定功能

![範例](<../.gitbook/assets/image (2) (2).png>)

### 使用方法:

對想要修改的信息右擊點選reply \
然後按以下格式輸入即可 \
`.edit 信息第一行` \
`信息第二行` \
`信息第三行`

{% hint style="info" %}
`使用前，`請確認自己擁有Admin 或頻道管理權限。
{% endhint %}
